Bubble Sort memiliki kompleksitas waktu O(n^2) hal ini disebabkan oleh cara kerja algoritma ini yang menggunakan 2 loop (nested loop) untuk mensortir elemen dalam array.
Loop 1 : Loop berjalan n - 1, dimana n adalah jumlah eleman array. karena setelah n - 1 pass maka array akan terurut
Loop 2 : Loop dalam melakukan perbandingan dan penukaran elemen.
